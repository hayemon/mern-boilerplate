# mern-boilerplate

MERN stack boilerplate project
On client side:
1. Webpack for bundling
2. Redux with Thunk for state management
3. Material UI for styles
4. CSS included in webpack config, but no templates present because css-in-js of Material Ui is used

Features present:
1. JWT Authentication: signup, signin, signout (back & front)
2. Authentication and validation middlewares (back)
3. Basic navigation bar and routing (client)
4. Basic template structure with CRUD (back & front)
